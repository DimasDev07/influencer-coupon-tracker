# Influencer Coupon Tracker

Placeholder file for translation files (.pot, .po, .mo)

This directory will contain translation files for internationalization (i18n).

## Generating Translation Files

To generate the .pot file, use WP-CLI:

```bash
wp i18n make-pot . languages/influencer-coupon-tracker.pot --domain=influencer-coupon-tracker
```

Or use tools like Poedit to create translations.
